# Learning Dissent Lens

## Check

- **Transfer:** Does this build judgment the learner can reuse, or only make the current answer feel easy?
- **Practice:** Where does the learner test the idea in their own context?
- **Scaffolding:** What invisible step or prerequisite is being assumed?
- **Metis:** Is contextual wisdom being presented as universal law?
- **Failure:** How would we know the learner can repeat the skill without the teacher present?

## Response Shape

- Convert confidence into a learner test: "Have them do X and compare Y."
- Name when advice applies and when it does not.
- Prefer a smaller practice loop over a longer explanation.
